## Why

Spec coding is becoming the next coding standard, and it is crucial for the development of AI agents. By defining clear requirements and acceptance criteria, we can ensure that our agents are built to meet specific goals and can be tested effectively. Additionally, implementing self-correction and self-improvement loops will allow our agents to learn from their mistakes and continuously enhance their performance.

## Core Goal

1. BDD Dirven Development + Spec (With Declear requirements boundaries and acceptance criteria)
2. Agent Self Correction Loops (via code linter and test feedback)
3. Agent Self Improvement Loops (via local keyword indexing and retrieval)
4. Automate Configure / Setup or guide user to do so (e.g. install dependencies, setup environment variables, etc.)
